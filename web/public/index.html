首次使用说明

接下来会怎么进行

当前 Web 部署在云服务器，Steam 登录需要借助你本机的 Chrome 插件完成回传。你不需要手动填写任何会话码。

1. 阅读并确认本说明
确认后系统会直接进入 Steam Remote Login 流程，不需要你再回主界面重复点击。
2. 安装 Chrome 插件
进入下一步后，页面会提供插件安装入口和教程。插件安装完成后会自动接管当前登录任务。
3. 只需要完成 Steam 登录
插件会自动打开 Rust+ Steam 登录页，自动读取 token 并回传到云端。
4. 云端自动同步
登录完成后，系统会自动同步 Steam 绑定信息，并继续后续的 Rust+ 配对监听。

认证完成后，你的 Steam 绑定摘要会记录到账号资料，用于后台管理、状态确认和问题排查。

免责声明：你需要确认自己有权使用对应 Steam / Rust+ 账号与服务器。系统会保存必要的登录凭据与绑定摘要，用于 Rust+ 功能运行和问题排查。未经授权的使用、共享账号、违规调用或对第三方服务造成的风险，由使用者自行承担。

确认前请先理解这套流程和风险。确认后会直接进入下一步，不再停留在当前说明页。
