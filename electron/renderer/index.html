🔗 设备配对
服务器配对流程
1
注册 FCM 推送通道
首次使用需要完成 Steam 账号登录授权
2
打开 Rust 游戏进入服务器
确保已在目标服务器中
3
游戏内按 ESC → Rust+ → Pair with Server
程序自动接收推送并保存服务器信息；若无新推送，请先在 Rust+ 官方 App 中取消旧配对再重新配对
